Design Guidelines

Visual Language & UI Components — Last updated: 2026-03-05

Color Palette

Our palette is professional, accessible, and balanced for both light and dark modes.

  • Primary Blue (#5E6AD2): Use for primary actions and brand presence.
  • Accent Orange (#F28C38): Use for status highlights and secondary CTAs.
  • Success Green (#22C55E): Use for positive states and completions.

Typography

We use DM Sans for body text and headers, and JetBrains Mono for technical data and code snippets.

  • Headers: Bold, tight letter-spacing for a modern feel.
  • Body: Regular/Medium, generous line-height (1.6) for readability.

Component Standards

All UI elements follow the **shadcn/ui** design patterns, modified for enterprise workflows.

  • Data Tables: Minimalist, sorting-enabled, with 12px row padding.
  • Drawers: Slide from right (30% width) for contextual details.
  • Cards: Subtle shadows (shadow-elevated) with 8px border radius.

Spacing & Grid

We follow a strict 4px/8px incremental spacing system to ensure consistent alignment and hierarchy.
