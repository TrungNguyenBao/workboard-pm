1. EXECUTIVE SUMMARY
Product Name

A-ERP: Agentic Enterprise Resource Platform

Vision

Tái định nghĩa ERP truyền thống thành một hệ thống vận hành thông minh, nơi:

Các module phối hợp qua MCP (Model Context Protocol)

Positioning

“ERP vận hành bằng AI Agent – không chỉ lưu trữ, mà chủ động tối ưu.”

7. CORE INNOVATION
7.1 MCP – Model Context Protocol

Chuẩn message nội bộ:

{
  "type": "event | command | context",
  "domain": "PMS | WMS | HRM | CRM",
  "payload": {},
  "permission_level": "read | suggest | action"
}

Giúp:

Agent giao tiếp chuẩn hóa

Context được chia sẻ realtime

Kiểm soát quyền ra quyết định
