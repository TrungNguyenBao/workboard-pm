HRM Product Requirements

Human Resource Management — Full Specification — Last updated: 2026-03-09

1. Tổng quan sản phẩm

Hệ thống Quản lý Hành chính Nhân sự (HCNS) nhằm mục tiêu chuẩn hóa 100% quy trình nhân sự, số hóa toàn bộ hồ sơ và biểu mẫu, sẵn sàng tích hợp ERP & AI Agent.

2. Phạm vi hệ thống

  • HR Core: Vòng đời nhân viên, tuyển dụng, nghỉ phép, chấm công, lương, hiệu suất.
  • Hành chính: Quản lý tài sản, mua sắm nội bộ, văn bản.
  • Tuân thủ: Quy định công ty, tuân thủ pháp luật lao động.

3. Core Data Models (32+ Tables)

Entity Key Fields Description
Employee user_id, name, email, department_id, hire_date Hồ sơ nhân viên
Department name, code, manager_id, parent_id Phòng ban & Sơ đồ tổ chức
LeaveRequest employee_id, type_id, start_date, status Yêu cầu nghỉ phép
PayrollRecord employee_id, period, gross, net, status Bảng lương hàng tháng
Recruitment title, quantity, requirements, deadline Quản lý tuyển dụng

4. API Endpoints

Hệ thống cung cấp REST API đầy đủ cho các module con:

  • /api/v1/hrm/employees - Nhân sự
  • /api/v1/hrm/departments - Phòng ban
  • /api/v1/hrm/leave-requests - Nghỉ phép
  • /api/v1/hrm/payroll-records - Tính lương

5. Frontend Pages (23+)

Hệ thống bao gồm Dashboard chiến lược, Danh sách nhân sự, Pipeline tuyển dụng (Kanban), Sơ đồ tổ chức trực quan (Org Chart) và các trang quản lý tài sản, mua sắm.
