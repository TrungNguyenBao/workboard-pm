    HR Quản lý Nghỉ phép Leave Management

    Hệ thống quản lý request nghỉ phép với approval workflow. Mỗi loại nghỉ có policy (tối đa ngày/năm, quy tắc), và tracking balance.

    Leave request system with approval workflow. Each leave type has policy (max days/year, rules), and balance tracking.

    Leave TypesLeave Types

    Annual Leave:Annual Leave: Nghỉ năm. Default: 12 ngày/năm (theo luật VN). Tích lũy qua nămVacation. Default: 12 days/year (VN law). Carries forward per policy
    Sick Leave:Sick Leave: Bệnh. Default: 30 ngày/năm (có giấy khám bệnh)Medical. Default: 30 days/year (with certificate)
    Unpaid Leave:Unpaid Leave: Không lương. Không tính vào lương. Cần approve managerUnpaid. No salary impact. Manager approval required
    Maternity:Maternity: Thai sản (theo luật: 4-6 tháng tùy trường hợp)Maternity (per law: 4-6 months depending)

    Leave Request WorkflowLeave Request Workflow

    1. Employee đăng ký request leave (chọn ngày bắt đầu, kết thúc, loại, lý do)Employee submits leave request (dates, type, reason)
    2. System check balance có đủ không. Nếu không đủ → rejectSystem checks balance. Reject if insufficient
    3. Gửi approval cho Line ManagerSend to Line Manager for approval
    4. Manager approve/reject. Nếu approve → status = approved, balance trừ điManager approves/rejects. If approved → balance deducted
    5. Payroll system update: mark days as leave trong attendancePayroll marks days as leave in attendance records
